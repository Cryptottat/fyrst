# FYRST -- 캐릭터/정체성

## 마스코트/캐릭터
- 이름: Sentry
- 종류: 강철빛 도베르만 (경비견)
- 설명: $FYRST 프로토콜의 수호자. 러그풀로부터 바이어를 지키는 존재.
- 성격: 단호하고 경계심 강함. 신뢰할 수 있는 유저에게는 꼬리를 살짝 흔듦.
- 디자인 규칙:
  - 미니멀 벡터 스타일 (플랫 + 약간의 그림자)
  - 강철빛 (스틸 그레이/블루) 피부톤
  - 목에 $FYRST 태그 목걸이
  - 한쪽 귀 접힌 상태
  - 앉아서 정면 주시 포즈
  - 눈동자에 은은한 블루 발광 (#2563EB)
  - 크기 비율: 16cm 기준
- 사용처:
  - 로고 옆 아이콘
  - Safe/Rug 알림 시 표정 변화
  - 로딩 화면
  - 에러 페이지
  - 트위터 프로필
- 금지 표현:
  - 의인화 (옷 입히기, 직립보행)
  - 귀여움/카와이 표현
  - 지나치게 사실적인 묘사

## 브랜드 톤
- 말투: 건조하고 단호함. 기관(institutional) 느낌. "우리가 해결한다"가 아닌 "시스템이 보호한다".
- 금지 표현:
  - 이모지
  - 과대광고 (MOON, 100x, guaranteed)
  - 밈코인 문화 슬랭 과용 (약간의 degen 용어는 허용)
- 허용 표현:
  - 보안/방어 관련 메타포 (fortress, shield, vault, sentry)
  - 데이터 기반 주장 (95% rug rate, $300M fees)
  - degen 공감 ("you've been rugged", "another day, another rug")

## 디자인 스타일
- Fortified: 방어적이고 견고한 느낌. 성벽, 방패, 금고 모티브.
- Institutional: 카지노(pump.fun) 대비 "신뢰할 수 있는 거래소" 느낌.
- Metallic: 금속 텍스처 (강철, 청동). 견고함과 불변성.
- Blueprint: 설계도 느낌의 라인 아트. 투명성과 구조적 안정성.

## 텍스처
- Brushed metal: 헤더/카드 배경에 미세한 금속 질감
- Blueprint grid: 배경 패턴으로 설계도 격자
- Subtle grain: 전체 오버레이로 미세한 그레인 노이즈
